비교 연산자

산술 연산자의 종류

산술 연산자 종류 설명
> a>b가 참이면 true, 거짓이면 false
< a<b가 참이면 true, 거짓이면 false
>= a>=b가 참이면 true, 거짓이면 false
<= a<=b가 참이면 true, 거짓이면 false
== num/=2는 num=num/2와 같음
!= a==b, a와 b가 같으면 true, 아닐시 false
==연산자의 경우 데이터 타입은 비교 X
=== a==b, a와 b가 같고 데이터가 같을시 true 아닐시 false
!== a!==b, a와 b가 같지 않거나 데이터 타입이 안같으면 true 아닐시 false

대입연산자 '='는 '같다'의 의미가 아닌 변수에 대입 (저장)을 의미
