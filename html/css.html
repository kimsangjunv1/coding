02. font-size, 단위의 고찰

글자 크기 명시 안할경우 16px

1. px

해상도에 따라 상대적으로 달라지는 기본 단위

font-size: 12px;

특별한 설정이 없다면 16px이 100% 기본, 150%는 24px가 됨. 부모 요소가 10px 일시 150%는 15px가 됨.

2.%

부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위

font-size: 150%;

특별한 설정이 없다면 16px이 100% 기본, 150%는 24px가 됨. 부모 요소가 10px 일시 150%는 15px가 됨.

3.em

부모 요소의 글자 크기를 100% 기준으로 계산한 100분의 1단위

font-size: 1.5em;

특별한 설정이 없을시 16px이 1em이 됨, 1.5em은 24px(16px*1.5=24px)이 되며 만약 부모요소가 10px라면 1.5em은 15px가 됨

4.rem

rem은 em과 비슷 부모가 아닌 최상위 요소의 크기를 100% 기준으로 계산. rem의 r은 root 즉, html요소를 뜻하며, html 요소에서 지정한 글자 크기가 1rem이 됨

font-size: 1.5rem;

5.vw, vh

vw는 뷰포트 너비값의 100분의 1단위, vh는 뷰포트 높이값의 100분의 1단위임

font-size: 10vw;

해설 : 뷰포트 너비의 10% 크기
